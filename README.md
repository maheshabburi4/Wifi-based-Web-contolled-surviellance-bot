# Wifi_based_Web_Controlled_Surviellance_Bot #

A bot connected through wifi and controlled via website used for surveillance

### Things required to build one ###
- Raspberry pi
- Web cam
- Toy car chassis
- Motors
- Wheels
- Motor driver (LC 2378)
- Bread board
- Connecting wires
- Battery
- Wifi Hotspot
- Phone or PC
